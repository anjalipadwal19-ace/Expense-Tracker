Expense Tracker Application
Project Description
This project is a web-based Expense Tracker developed using HTML, CSS, and JavaScript.
It helps users record their income and expenses, view balance, and analyze spending using charts.
The application stores data in the browser, so user information is saved even after refreshing the page.
________________________________________
Files Included
•	index.html – Main structure of the application
•	ET.css – Styling and layout of the application
•	ET.js – Handles all functionality and logic
•	README.md – Project explanation and usage guide
________________________________________
How to Use the Application
1.	Open index.html in any modern web browser.
2.	Enter your name in the sidebar and press Enter.
3.	Use the sidebar to navigate between:
o	Home
o	Income
o	Expense
o	Overview
4.	Add income or expense details using the forms.
5.	View recent transactions, balance, and charts on the Home page.
6.	Use Edit or Delete options to manage transactions.
7.	You can export or import income and expense data as JSON files.
8.	Click Logout to switch user.

Features of the Project
•	Separate sections for Income and Expense
•	Automatic calculation of:
o	Total Income
o	Total Expense
o	Balance
•	Monthly report table
•	Pie chart for expense categories
•	Line chart for monthly income and expense trend
•	Edit and delete transactions
•	Export and import data option
•	Profile name and avatar support
•	Data stored using browser local storage

Explanation of Code Structure
HTML
HTML is used to create:
•	Sidebar navigation
•	Forms for adding income and expenses
•	Tables for transactions
•	Sections for home, income, expense, and overview
•	Charts using Canvas
CSS
CSS is used for:
•	Layout design using Flexbox and Grid
•	Styling cards, tables, buttons, and forms
•	Sidebar and responsive layout
•	Making the application look clean and user friendly
JavaScript
JavaScript handles:
•	Adding, editing, and deleting transactions
•	Saving data in local storage
•	User profile handling
•	Calculating totals and balance
•	Generating charts using Chart.js
•	Importing and exporting data
•	Updating UI dynamically
